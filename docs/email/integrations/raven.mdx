Raven Email integration is a test integration. Users will get 100 free credits when they signup on the Raven platform. 
You can use these free credits to test the platform. If you need more credits please mail to support@ravenapp.dev.

### Raven Email Integration in Raven

| Keys            | Details                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Name`       | Give any unique name to the integration                                    |
| `From Name`     | From Name is the sender's email address (optional)   |
| `Reply To`     | Reply To is the recipient's email address (optional)  |
