---
title: "Get User Profile"
description: "Gets the requested user profile"
api: "GET https://api.ravenapp.dev/v1/apps/{app_id}/users/{user_id}"
---

### Parameters

<Param path="app_id" type="string" required>
  Your AppId
</Param>

<Param path="user_id" type="string" required>
  Your user identifier
</Param>

<RequestExample>

```json Request Example
GET /v1/apps/{{app_id}}/users/{{user_id}} HTTP/1.1
Host: https://api.ravenapp.dev
Authorization: AuthKey {{secret_key}}
```

</RequestExample>
