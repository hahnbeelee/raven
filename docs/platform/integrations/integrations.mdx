---
title: "Integrations"
description: "Integrations are channels providers which you can use to send notifications."
---

To get started with a provider, all it takes is to add your provider keys to Raven.
This removes any effort needed to integrate or manage provider APIs and also
allows you to switch anytime to a different provider.

Head over to the [Integrations tab](https://console.ravenapp.dev/get-started)
where you will see list of available providers (for every channel). Select a
provider and add its keys to make it live.

![Select Provider in Integrations Tab](/images/select-provider.gif)

Our channel wise Integration doc will help you add your provider.

<Note>
  Your Integration keys are encrypted and stored securely. Check our security
  doc for more details on how we store them.
</Note>
